常用的通函数及 API

发表 2020-05-02 更新 2020-08-09 分类 数据分析 字数： 4k

numpy 是 python 的强大科学计算库, 它让 python 处理数据, 进行科学计算变得极其便捷, 高效

五, 常用的通函数及 API

numpy 调用, 用来操作 ndarray 对象的函数

1, 修改数组形状

reshape 与 ndarray 对象的方法一样, 另有 resize 只能按行展开

array([[4, 2, 3],
       [1, 0, 0]])

array([[4, 1, 2, 0, 3, 0]])
array([[4, 2, 3, 1, 0, 0]])

ravel 展开为一维(ndarray 对象也有该方法)

array([2, 1, 1, 2, 4, 3])

2, 组合数组

concatenate, hstack, vstack 连接数组

array([[2, 1, 4, 3, 0, 1],
       [1, 2, 3, 3, 2, 1]])
array([[2, 1, 4, 3, 0, 1],
       [1, 2, 3, 3, 2, 1]])
array([[2, 1, 4],
       [1, 2, 3],
       [3, 0, 1],
       [3, 2, 1]])

stack 堆叠数组, 堆叠后维度增加 1

(array([[2, 1, 4],
        [3, 0, 1]]),
 array([[2, 3],
        [1, 0],
        [4, 1]]))

3, 拆分数组

split, hsplit, vsplit 将数组拆分, 返回为列表

([array([[2, 1, 4]]), array([[1, 2, 3]])],
 [array([[2, 1, 4]]), array([[1, 2, 3]])],
 [array([[2],
         [1]]),
  array([[1],
         [2]]),
  array([[4],
         [3]])])

4, 添加, 删除元素

delete 删除指定索引的元素, 行或列

(array([[2, 1, 4],
        [1, 2, 3]]),
 array([2, 4, 1, 2, 3]))
(array([[2, 1, 4]]),
 array([[2, 4],
        [1, 3]]))

insert 在指定索引前插入元素, 行或列

array([2, 9, 1, 4, 1, 2, 3])
(array([[2, 1, 4],
        [9, 9, 9],
        [1, 2, 3]]),
 array([[2, 1, 4],
        [1, 2, 3],
        [1, 2, 3]]))
array([[2, 1, 1, 4],
       [1, 3, 2, 3]])

append 在数组末尾(或行,列末尾)加入元素, 行或列(维数必须相同)

array([2, 1, 4, 1, 2, 3, 2])
(array([[2, 1, 4],
        [1, 2, 3],
        [2, 2, 2]]),
 array([[2, 1, 4, 2],
        [1, 2, 3, 2]]))

unique 查找数组中的唯一值

array([[2, 3, 2],
       [2, 2, 0]])

(array([0, 2, 3]),
 array([5, 0, 1], dtype=int64),
 array([1, 2, 1, 1, 1, 0], dtype=int64),
 array([1, 4, 1], dtype=int64))

5, 重新排列元素

flip, fliplr, flipud 翻转数组

(array([[0, 2, 2],
        [2, 3, 2]]),
 array([[2, 2, 0],
        [2, 3, 2]]),
 array([[2, 3, 2],
        [0, 2, 2]]))

sort 排序数组元素

array([[2, 2, 3],
       [0, 2, 2]])

roll 滚动数组元素

(array([[0, 2, 3],
        [2, 2, 2]]),
 array([[2, 2, 0],
        [2, 3, 2]]),
 array([[2, 2, 3],
        [0, 2, 2]]))
