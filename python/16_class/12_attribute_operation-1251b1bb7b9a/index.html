属性操作

Python 是动态语言，对象的属性不仅可以查看、访问、调用，还可以动态地增、删、改。

给类 A 及其实例动态添加属性，属性可以是数据属性和方法。类层级添加的属性，将作为所有实例的属性；实例添加的属性，只有对应的实例能访问。
